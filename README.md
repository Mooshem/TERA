# TERA
The Earth Restoration Adventure

# Our Project
TERA is a social media platform to collaborate with your community in cleaning up the Earth. The platform aims to bring people together in a fun and beneficial manner, by earning points and badges regarding your contributions to the planet. TERA will allow users to post community meet-ups, where others will be able to show up and participate in group restoration activities. 

# Tech Stack

- React Native (Expo)
- TypeScript (TSX)
- Firebase (Authentication + Firestore)

---

# Setup Instructions

## 1. Clone the repository

```bash
git clone https://github.com/your-username/tera.git
cd tera
```

## 2. Install dependencies
```bash
npm install
npm install firebase
```

## 3. Start the development server
```bash
npx expo start
click 'w' to open in web
```
